# crawl_thread
## Env
```bash
$ conda create -n crawler python=3.10.14
```
## Activate
```bash
$ conda activate crawler python=3.10.14
```
## Installation
```bash
$ pip install -r requirements.txt
```
## Run
```bash
make USERNAME=<your_thread_username> PASSWORD=<your_thread_password> NUMBER=<number_reload_thread> 
```
